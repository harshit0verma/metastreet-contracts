# MetaStreet Contracts

## Usage

Install:

```
npm install
```

Compile contracts:

```
npx hardhat compile
```

Run unit tests:

```
npx hardhat test
```

Start hardhat network:

```
npx hardhat node
```

Deploy simulation environment (after starting hardhat network in another shell):

```
npx hardhat run --network localhost scripts/deploy-simulation.ts
```
